Build it, code it, watch it work.

Robotics

Three tracks

Three levels that build serious skill through serious projects. Children move up only when they are ready, not when the year ends.

01AGES 7-9

Junior · LEGO Mindstorms

First circuits, first code blocks, first robot that follows a line. Curiosity becomes a working machine.

02AGES 10-13

Builder · Arduino

Real sensors, real wiring, real bugs to fix. Children move from blocks to written code, one careful step at a time.

03AGES 14-16

Maker · Python

A full term project. Children pick the problem, design the build, ship a working prototype, and present it.
